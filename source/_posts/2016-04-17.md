### 前言
什么? 你已经很了解python了? 好.
我们开始做个测验. 下面有12个题目. 填个空, 让这个代码片段变的可能.
### 先做个例子
例子1:

``` python    
    
    >>> x, y = ???  
    >>> x + y == y + x  
    False  
      
```
  
其中有个`???`. 把它替换成什么就可以让这段代码成立呢?
揭晓答案:

``` python    
    
    >>> x, y = [0], [1]  
    >>> x + y == y + x  
    False  
      
```
  
这个还是蛮简单的. 好继续
例子2:

``` python    
    
    >>> x = ???  
    >>> x < x  
    True  
      
```
  
答案呢?
如果使用python内置的数据结构, 确实是不可能的. 唯一可以`硬造`一个例子

``` python    
    
    >>> x = type("", (), {"__lt__": lambda a, b: True})()  
    >>> x < x  
    True  
      
```
  
但是这里不需要这么复杂的去想`trick`出结果, 只考虑python已经实现的设计
